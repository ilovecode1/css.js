#5 Reasons To Use CSS.js:

1. Simple!

2. Less Code!

3. Live!

4. Fast!

5. Great For Animations!

#About

Simply add CSS with JS!

```
cssjs("yourcsscode");
```

It updates live so you do not have to worry!

#Demo

HTML

```
<script src="path/to/css.js"></script>
<h1>CSS.js</h1>
```

JS

```
cssjs("h1{background: red}");
```
